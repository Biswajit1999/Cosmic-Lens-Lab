# Differentiators

CosmicLens Lab should not merely copy existing Python lensing packages into a browser. Its strongest differentiators are the combined product experience.

## 1. Subhalo anomaly playground

Users inject small perturbers and immediately see local image residuals, caustic perturbations, and flux-ratio changes.

## 2. Model-mismatch inspector

The UI highlights when external shear is compensating for missing internal mass flexibility.

## 3. Browser-to-Python reproducibility

Every scene can be exported to JSON and checked with Python validators.

## 4. Time-delay cosmography sandbox

A toy but honest view of how Fermat potential, mass profile, and H0 interact.

## 5. Survey-mode generator

Batch-generate mock lenses with PSF, noise, masks, and metadata for education and algorithm testing.

## 6. Teaching mode

A guided sequence: point mass → SIS → SIE → composite model → substructure → time delay.
