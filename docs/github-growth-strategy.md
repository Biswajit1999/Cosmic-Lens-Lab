# GitHub growth strategy

## Positioning

CosmicLens Lab should be introduced as:

> A browser-native gravitational-lensing laboratory for students, researchers, and scientific-visualisation builders.

## Launch assets

- short demo GIF or video of slider-to-caustic interaction
- README banner
- one-click GitHub Pages demo
- benchmark gallery
- physics documentation page
- Python validation notebook

## Star-attraction checklist

- Add a polished social preview image.
- Pin the repo on the GitHub profile.
- Add topics: `gravitational-lensing`, `astrophysics`, `cosmology`, `webgpu`, `scientific-visualization`, `typescript`, `python`, `education`.
- Create issues tagged `good-first-issue`.
- Make demos shareable through URLs.
- Publish a LinkedIn launch post with a 20-second screen recording.
- Submit to astronomy/software communities only after validation docs are ready.

## Non-negotiables

Do not overclaim precision cosmology. Do not imply replacement of lenstronomy or PyAutoLens. Frame the repo as an interactive bridge between education, visualisation, and reproducible scientific prototyping.
