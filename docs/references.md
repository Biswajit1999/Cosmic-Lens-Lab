# References and reading map

This project is built around standard gravitational-lensing literature and modern open-source practice.

## Foundational lensing

- Schneider, Ehlers & Falco — *Gravitational Lenses*.
- Blandford & Narayan — gravitational lensing review literature.
- Narayan & Bartelmann — lectures on gravitational lensing.
- Keeton — computational methods for gravitational lenses.

## Canonical models

- Kormann, Schneider & Bartelmann — singular isothermal ellipsoid.
- Navarro, Frenk & White — universal dark matter halo profile.
- Bartelmann — NFW lensing behaviour.
- Wright & Brainerd — lensing by NFW haloes.
- Cardone — lensing properties of circular Sérsic models.

## Modern context

- Time-delay cosmography reviews and H0LiCOW/TDCOSMO-era modelling discussions.
- Literature on line-of-sight shear, model flexibility, and strong-lens systematics.
- Open-source packages: lenstronomy, PyAutoLens, GLAMER, and related strong-lensing modelling tools.

## Software references

- Vite and TypeScript for static scientific web apps.
- Vitest and Playwright for unit and visual regression testing.
- Astropy cosmology for Python reference distances.
- WebGPU for future compute kernels with WebGL/Canvas fallbacks.

## Citation policy

Whenever adding a model, add:

1. original paper or canonical derivation,
2. numerical method reference where applicable,
3. convention notes,
4. validation tolerance.
